# ShellOrchestra frontend

Vite React SPA using TanStack Router, TanStack Query, MUI, and an OpenAPI-generated TypeScript client.

Passkey browser ceremonies use `@simplewebauthn/browser`; API calls stay behind the local OpenAPI client boundary where possible.

The frontend source is published for inspection and security review under the ShellOrchestra Source Available License. Official production builds are generated by the official Develastic release pipeline and shipped as signed release artifacts.
