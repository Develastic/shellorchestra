# OS and distro SVG icons

These local SVG assets are used to display server operating system and Linux distribution badges.

Sources:

- Most distro icons and generic Linux fallback: Simple Icons, CC0-1.0.
  - Project: https://github.com/simple-icons/simple-icons
  - Slugs list: https://github.com/simple-icons/simple-icons/blob/develop/slugs.md
  - Disclaimer: https://github.com/simple-icons/simple-icons/blob/develop/DISCLAIMER.md
  - Local SVG slugs currently include: `almalinux`, `alpinelinux`, `archlinux`,
    `centos`, `debian`, `endeavouros`, `fedora`, `gentoo`, `kalilinux`, `linux`,
    `linuxmint`, `manjaro`, `nixos`, `opensuse`, `popos`, `proxmox`, `raspberrypi`,
    `redhat`, `rockylinux`, `suse`, and `ubuntu`.
- Windows icon: Bootstrap Icons, MIT License.
  - Project: https://github.com/twbs/icons
  - License: https://github.com/twbs/icons/blob/main/LICENSE
