# ShellOrchestra app icon assets

`shellorchestra_prompt_icon_1.png` and its exact 16/32/48 px companions are the simplified prompt-in-a-square icon used below 64 px and for the favicon.

`shellorchestra_icon_1.png` is the full-size square application icon source image used from 64 px through 119 px.

`shellorchestra_logo_1.png` is the cropped full-size application logo source image used for generated public icon/logo sizes from 120 px upward.

Generated favicon, Apple touch, and PWA sizes live in `frontend/public/`. Official generated assets are produced by the official Develastic release tooling.
